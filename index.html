SIG-TLS-002 / Chronolocation Calculator

The sun is a witness. Make it testify.

Verify the date, time, and location of a still image or video using solar position. Enter coordinates, date, and time. Optionally measure a shadow. Returns sun altitude, azimuth, sunrise, sunset, and a citable chronolocation report.

±0.1° Solar position accuracy
15% Shadow tolerance
SS-DOSSIER Report standard
v0.1 Tool version

Date and time ISO 8601
Enter the local UTC offset if the time above is local. Example: +2 for CEST, -5 for EST. Leave 0 if the time is already in UTC.
Shadow verification Optional
Include shadow-length check
Vertical height of the object casting the shadow.

Set coordinates and a moment in time. The sun will tell you the rest.
